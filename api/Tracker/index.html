Tracker

A tracker is a class that always display (or tracks) given parts or vectors. It has a prebuilt magnitude, unit and min-max range. Unlike the other classes above, you might want to change some of its properties before starting.

note

You can do self:Function() even if this document says Tracker.Function(self)!
