CanSeeTarget​

Radius Function
</>
utility.CanSeeTarget(
Position: Vector3 | CFrame,--

The position that the raycast will start on.

Target: BasePart | Model,--

The part or model that should be checked.

RayCastParams: RaycastParams?,--

Optional params.

DotProduct: number?,--

The minimum dot product it should have.

Distance: number?--

The maximum distance it should have.

) → boolean

Cast a raycast and returns a boolean indicating if the ray hit the basepart or anything inside the model. If a dot product is provided, it will check if it's facing the part too, the dot product goes from -1 (looking backwards) till 1 (looking directly at the part). If distance is provided, then it will also check if the distance is at least the number provided.

FilterTable​

Table Function
</>
utility:FilterTable(
t: {T},--

The table that should be filtered.

callBack: (value: T) → boolean--

The callback function that determines wheter to filter the certain value or not. MUST return a boolean.

) → {any}

For every value in the table, the callback will fire with the value as the parameter. The callback MUST return a boolean. If the boolean is true on that value then it'll add to the returning table, else ignore.
